# Info Vac
**Seja bem vindo!**
Esse protejo foi desenvolvido para a instituição Irmã Adelaide com o propósito de fazer concentralização sobre vacinas! Também deixo o código aqui caso ele possa te ajudar de alguma forma! Sinta-se livre para baixar e modificar o que bem entender!
## Ferramentas utilizadas nesse projeto
Fimga - Layout base
Android Studio - SourceCode e XML layout
**Todas** as ilustrações foram retiradas do site Storyset.com
## Instalação
O app é leve e simples! O .apk está na pasta *Instalador/infovac.apk* e basta executar ele em seu aparelho para instalar.
